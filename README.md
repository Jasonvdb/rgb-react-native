# rgb-react-native
Experimental react native app using RGB-SDK

### Installation

- `git clone https://github.com/Jasonvdb/rgb-react-native.git`

- `cd rgb-react-native`

- `yarn install`

- `cd ios && pod install && cd ../`

- open `ios/RgbReactNative.xcworkspace` and run from Xcode to see runtime errors.
